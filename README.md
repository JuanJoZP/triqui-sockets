# Triqui (Tic-Tac-Toe)

Web server to play Tic-tac-toe, you can play against another person or against the AI.

## Installation

clone the github repository and run the following

```
python ai_service
node server.js
```

then go to [localhost](http://localhost:3000) and you can start to play.
